# Acceptance

Define "done" before the work starts, so output can be checked instead of trusted
on faith.

## Done Means

The concrete condition that has to be true. Start the line with "done means ...".

## Evidence Is

The proof you would point to: a test that passes, a recording, a file, a
checked-in artifact.

## Still Not Done If

The failure conditions that keep this open even when it looks finished.

---

中文要点：动手前先把"做完"定义清楚——做完意味着 / 证据是 / 以下情况仍未完成——让产出可被
检查，而不是凭感觉信任。

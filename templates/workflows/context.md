# Context

Capture the real boundary of the task so the assistant does not produce plausible
work that misses the point.

## Goal

What outcome are we after? One sentence.

## Audience / User

Who is this for? Who will use or read the result?

## Current State

What exists now? What is already decided, drafted, or built?

## Constraints

Time, budget, tools, dependencies, and any hard limits.

## Non-Goals

What is explicitly out of scope this round?

---

中文要点：用五行写清任务边界——目标 / 受众 / 现状 / 约束 / 非目标——避免助手朝错的方向
产出"看着合理、其实跑偏"的东西。
